# Direct, Resolver, and M43-A Parity Evidence

One-step exact terminal distributions were compared for these effect families:

- side-filtered Exalted;
- Greater plus side-filtered Exalted;
- side-filtered Annulment;
- side-filtered Chaos removal;
- side-filtered Perfect Essence removal.

For every case, direct execution of the resolver-produced operation and one-step M43-A execution produce identical canonical terminal hashes and exact rational masses.

M43-A resolves active modifiers against the actual current branch state. It retains the explicit accepted base-operation executor registry; the Omen layer cannot register a new base operation or bypass currency admission.
