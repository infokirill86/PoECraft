# Compatibility Matrix

| Base accepted operation group | Add-count effect | Add-side effect | Removal-side effect | Removal selector | Allowed composition |
|---|---|---|---|---|---|
| Exalted | Greater Exaltation | Sinistral or Dextral Exaltation | none | none | add-count plus at most one add-side |
| Annulment | none | none | Sinistral or Dextral Annulment | none | one removal-side effect |
| Chaos | none | none | Sinistral or Dextral Erasure | Whittling | removal-side plus selector |
| Perfect Essence | none | none | Sinistral or Dextral Crystallisation | none | one removal-side effect |

Rules enforced by code:

- duplicate Omen IDs fail closed;
- two effects in the same dimension fail closed;
- an Omen targeting another operation group fails closed;
- unknown, unavailable, historical, blocked, or non-admitted IDs fail closed;
- cross-dimension composition exists only inside the pinned groups above.
