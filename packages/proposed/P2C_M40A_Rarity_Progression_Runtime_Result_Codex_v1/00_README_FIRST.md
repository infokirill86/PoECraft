# P2C M40-A Rarity Progression Runtime Result — Codex v1

Package type: `IMPLEMENTATION_RESULT_DELTA / PROPOSED`

M40-A implements one shared data-driven single-add family for ten authorized catalog rows:

- Transmutation, Greater Transmutation, Perfect Transmutation;
- Augmentation, Greater Augmentation, Perfect Augmentation;
- Regal, Greater Regal, Perfect Regal;
- base Exalted.

The result is proposed for Claude audit. It is not self-accepted runtime or project truth.

Augmentation and base Exalted use the same executor without a rarity change.
