# Risks and deferred items

## Review risks

- Rarity transition is the first accepted-candidate runtime that changes rarity; Claude should inspect isolated working-state and atomic rollback behavior closely.
- `runtime_admission_status` is set in this proposed delta as authorized, but M40-A is not accepted truth until a later User gate.
- Source evidence is project-model/source-open and may need later updates when trusted external wording changes.
- The shared executor is intentionally narrow; pressure to make it a general operation algebra should be rejected.
