# Plain-language summary for Kirill

M41-A teaches the simulator one new type of craft: a Greater Essence on a Magic quarterstaff. Unlike an Exalted-style craft, this does not roll among many possible modifiers. The selected Essence names one exact modifier, so the engine verifies that modifier against the central modifier database, changes the item from Magic to Rare, and installs that exact modifier as one atomic action.

If any check fails, nothing happens: the original item is unchanged and the currency is treated as not consumed. Existing modifiers and fractured flags are preserved.

Only the eight authorized Greater Essences were activated. Perfect, Lesser, and Corrupted Essences remain closed. The broader question of how many Essence-crafted modifiers an item may hold is still unverified and was not silently decided here.
