# Participant and architecture critique

No material objection remains after the gate clarification.

The original design audit correctly identified crafted capacity as source-open. Blocking the Greater-only wave on the complete general rule would be unnecessary because this implementation does not define stacking, replacement, multiple-Essence, or Perfect-Essence behavior. It uses the existing shared item-state validator and labels the unresolved rule explicitly.

The selected boundary is right-sized: eight rows share one deterministic transition shape and one canonical-data path. Splitting them into eight milestones would create ceremony without improving safety; adding other Essence tiers would require mechanics decisions that are not authorized.
