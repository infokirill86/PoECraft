# Numeric Quarantine Report

## Public result files

The public result files intentionally avoid probability-value publication.

They may mention:

- seed presence;
- sample-count presence;
- tolerance logic;
- pass or fail status;
- file names;
- hashes;
- boundaries.

They do not include:

- observed Monte Carlo estimates;
- exact probability values;
- percentages;
- decimal probability renderings;
- rational probability renderings;
- Python fraction-constructor renderings;
- success chance tables.

## Quarantined artifacts

No separate quarantined numeric artifact was produced for M33.

Reason: the required validation is represented by tests and pass/fail reports. The package does not need to carry observed probability evidence.
