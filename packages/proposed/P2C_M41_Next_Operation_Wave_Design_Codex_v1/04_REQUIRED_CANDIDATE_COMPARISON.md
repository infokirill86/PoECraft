# Required Candidate Comparison

| Candidate | Product value | Source/mechanics readiness | Shared accepted architecture | Decision |
|---|---|---|---|---|
| Alchemy / multi-add | Adds an alternative Normal/Magic-to-Rare route | Tooltip/data support four modifiers, but sequential generation and discard-all semantics need a separate proof | M40 target-rarity state plus repeated ordinary-add pool rebuild | Defer; coherent later wave, less direct to primary fractured-Rare route |
| Essence family | Adds guaranteed modifier control; Abrasion is directly physical | Greater core confirmed; Perfect removal/capacity edge unresolved | Guaranteed-add executor; Perfect later adds shared removal branch | **Select Greater core for M41-A; split Perfect** |
| Omen/modifier layer | High directional-control value on accepted currencies | Individual rules mostly documented, but current PoE2DB contains Omens absent from repo data; Whittling tie behavior remains project policy | M38 resolver filter dimensions, ordinary add/removal kernels | Defer pending Omen inventory/source refresh; do not batch heterogeneous effects blindly |
| Fracture | Creates protected modifiers | Clean core exists, but revealed-Desecrated eligibility remains disputed | Shared instance selection plus flag mutation | Defer; primary project start is already fractured and the disputed edge must stay gated |
| Jawbone / Desecrated / Reveal | Important future targeted crafting | Highest mechanics and source complexity; PD-013 remains open | Existing placeholder and reveal-pool scaffolding | Defer as a separate major wave |
| Longer chains / search preparation | Improves route execution breadth | Technically feasible for admitted operations | Existing exact/MC and replay infrastructure | Defer; operation surface is not yet rich enough to justify more chain infrastructure |

## Additional candidate: support augment / Astrid

The repo contains an `install_astrid` admission candidate. It is narrower than Essence and does not add the guaranteed-modifier capability needed by the original crafting model, so it is not selected for M41.
