# P2C M41 Next Operation Wave Design

Status: **PROPOSED DESIGN — NOT RUNTIME ADMISSION**

M41 selects the Greater Essence quarterstaff family as the widest currently safe next implementation wave. The proposed M41-A batch contains eight Greater Essence rows already present in project data. They all upgrade a Magic quarterstaff to Rare and install one specific guaranteed crafted modifier through one shared executor.

The broader Essence family is intentionally split. Perfect Essence is more directly useful to the fixed fractured-Rare starting lane, but its removal/capacity interaction is not fully specified by trusted wording. It must not be implemented until that rule is verified or explicitly adopted as project policy.

This package changes no runtime, data, operation admission, accepted truth, or boundary status.
