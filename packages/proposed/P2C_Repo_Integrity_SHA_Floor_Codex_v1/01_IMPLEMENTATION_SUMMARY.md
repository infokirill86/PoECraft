## How the updater works

`tools/update_sha256sums.py`:

- finds the git repository root;
- lists tracked files plus untracked non-ignored files using git;
- excludes the root `SHA256SUMS.txt` itself;
- sorts repository-relative POSIX paths deterministically;
- writes the root `SHA256SUMS.txt` with SHA-256 hashes.

The root checksum file is now meant to be generated by the tool, not edited manually.
