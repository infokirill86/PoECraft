# Plain-Language Summary for Kirill

The project is ready to design Desecration, but not yet ready to implement it honestly.

The reliable game contour is clear: a Jawbone applies a hidden Desecrated modifier to a Rare quarterstaff, that hidden modifier occupies a prefix or suffix slot, and the Well of Souls later offers a choice of three replacements. Ancient Jawbone applies the prepared minimum-level rule. An item normally cannot be Desecrated again while either hidden or revealed Desecrated state remains.

The exact random process is not fully published. Current sources do not settle how the side is chosen when both sides are available, what happens when only one side is full, or the exact weighted algorithm used to build the three Reveal offers. The repository already contains answers, but those answers are prepared project policies, not source-proven runtime truth.

Fracture has a separate conflict. Current wiki documentation says Desecrated modifiers count toward Fracture's minimum but cannot themselves be fractured. The repository also preserves an older Craft of Exile observation in which a revealed Desecrated modifier was eligible. M47 does not choose between them.
