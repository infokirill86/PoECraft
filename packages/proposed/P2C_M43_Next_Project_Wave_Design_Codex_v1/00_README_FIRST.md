# P2C M43 Next Project Wave Design

Status: **PROPOSED DESIGN — NOT RUNTIME AUTHORIZATION**

M42-A is already accepted and published as the starting point for this design. This package does not duplicate that acceptance record.

M43 selects the next project wave as **bounded fixed-sequence evaluation over already accepted operations**. The project now has enough operation breadth that its main product bottleneck is composition: it can execute many useful operations individually, but the accepted heterogeneous-chain runtime remains fixed to a narrow two-step surface.

The proposed M43-A floor would execute a caller-supplied sequence of admitted operations through the existing resolver and shared exact/Monte Carlo kernels. It would not choose a route, rank alternatives, recommend actions, admit Omens, or add any new crafting mechanic.

No runtime, tests, operation admission, data semantics, accepted truth, public numeric output, optimizer, economics, automation, or boundary status changes in this design package.
