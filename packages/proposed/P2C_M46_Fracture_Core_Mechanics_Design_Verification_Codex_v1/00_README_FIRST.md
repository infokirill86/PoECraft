# M46 Fracture Core Mechanics Design Verification

Package type: `DESIGN / MECHANICS_VERIFICATION`

This package proposes a clean base Fracture contract for a later separately gated M46-A implementation. It implements no runtime, admits no operation, publishes no numeric probability, and closes no SOURCE/PROVENANCE, MML, crafted-capacity, or PD-013 question.

The proposed clean core is deliberately useful for the physical-quarterstaff route while failing closed on Desecrated/Revealed and other disputed special-state interactions.

Fresh external source verification was performed on 2026-07-12. See `03_SOURCE_AND_REPO_COMPARISON.md` for exact sources, findings, and limitations.
