## Broadest coherent batch

The coherent unit is not one Omen per milestone. It is one resolver modifier compiler plus a pinned set of effects over already accepted operations:

- add-count override;
- add-side filter;
- removal-side filter;
- minimum-modifier-level removal selector.

These dimensions share admission, compatibility, fail-closed, trace, exact, and replay rules. Effects requiring Desecrated selection, unaccepted operations, historical-version policy, or an unresolved multi-remove probability shape are not part of this batch.
