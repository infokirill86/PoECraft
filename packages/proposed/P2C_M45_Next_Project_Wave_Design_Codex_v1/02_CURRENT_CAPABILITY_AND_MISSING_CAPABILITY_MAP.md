# Current Capability and Missing-Capability Map

## Accepted capability surface after the M44-A gate

| Capability | Current accepted state |
|---|---|
| Ordinary weighted add | Accepted shared engine primitive |
| Rarity progression | Base/Greater/Perfect Transmutation, Augmentation, and Regal; base Exalted |
| Removal | Base Annulment with fractured protection |
| Remove then add | Base/Greater/Perfect Chaos-like operations |
| MML variants | Greater/Perfect Exalted and Chaos plus accepted rarity-progression variants |
| Deterministic Essences | Eight Greater Essence quarterstaff rows |
| Remove plus guaranteed Essence | Six Perfect Essence quarterstaff rows under the temporary crafted-count floor |
| Atomic multi-add | Base non-fractured quarterstaff Alchemy with four sequential accepted adds |
| Sequence evaluation | Caller-supplied fixed sequences of one through eight accepted operations; exact ceilings and seeded replay |
| Operation compilation | Single-operation resolver with fail-closed admission |

## Material missing capabilities

| Gap | Product consequence | Current boundary |
|---|---|---|
| Independent currency modifiers / Omens | Cannot express side-directed add/removal or Whittling control used around the route | No Omen is runtime-admitted |
| Fracture creation | Cannot model acquisition of a fractured base | Primary scope intentionally starts from a purchased fractured base |
| Essence repeat/replacement | Cannot model repeated Essence crafting | Crafted-capacity contract remains source-open |
| Desecrated/Jawbone/Reveal | Cannot model the staged special-modifier route | Separate high-risk contour; PD-013 remains open |
| Conditional route search | Cannot discover or recommend a strategy | Deliberately outside evaluator scope; planner/optimizer remains closed |
| Public probability presentation | Internal results cannot be released broadly | Separate release gate remains closed |

The missing Omen layer is the largest truth-neutral architectural gap that also has immediate route value.
