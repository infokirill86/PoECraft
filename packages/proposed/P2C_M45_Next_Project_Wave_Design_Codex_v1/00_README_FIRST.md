# P2C M45 Next Project Wave Design

Package type: `DESIGN / MECHANICS_VERIFICATION`

This package records the M45 project-frontier decision after acceptance of M44-A base Alchemy. It proposes an independent Omen modifier layer over already accepted operations as the highest-value next wave.

This is design only. It admits no Omen, changes no runtime behavior, publishes no numeric probability, and closes no SOURCE/PROVENANCE, MML, crafted-capacity, or PD-013 question.

The proposed later M45-A floor is deliberately limited to a source-aligned clean core. Historical/drop-disabled Omens, Desecrated/Jawbone/Reveal effects, multi-remove ambiguity, and any new base operation remain separately gated.
