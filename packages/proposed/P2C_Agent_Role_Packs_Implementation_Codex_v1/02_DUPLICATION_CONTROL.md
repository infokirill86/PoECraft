# Duplication Control

Claude's main refinement was to avoid copying the same doctrine into every file.

This implementation follows that rule:

- `manifest/Participant_Voice_Charter.md` remains the detailed participant-voice doctrine.
- `manifest/GitHub_Workflow_Protocol.md` remains the detailed workflow/dispatcher/checksum doctrine.
- `manifest/Operating_Manifest_v4.md` remains the operating baseline.
- `manifest/Agent_Role_Pack.md` is thin and mostly points to those accepted files.
- `AGENTS.md` contains only Codex-specific read order, role, hygiene, and output format.
- `CLAUDE.md` contains only Claude-specific read order, role, audit expectations, and escalation triggers.

The new files do not copy full milestone history and do not become ledgers.
